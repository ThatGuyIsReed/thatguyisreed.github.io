# thatguyisreed.github.io

## Structure

- `index.html`: Home page with icon, YouTube link, and featured mods.
- `pages/`: Multi-page site (`about`, `projects`, `skills`, `contact`, `donate`).
- `pages/projects/`: Project detail pages.
- `assets/css/styles.css`: Shared styling.
- `assets/images/profile-icon.svg`: Replace this with your real icon image.
- `boilerplate/`: Reusable commented starter template.
