# Boilerplate Starter

This folder is a reusable starter you can copy for future static portfolio/site projects.

## What is included

- `site-template/index.html`: Base page with commented structure.
- `site-template/assets/css/styles.css`: Base variables/layout with comments.
- `site-template/assets/js/main.js`: Starter script with comments.
- `site-template/config.example.js`: Example runtime config.

## Setup

1. Copy `boilerplate/site-template` to a new folder.
2. Rename `config.example.js` to `config.js`.
3. Add only public config values (handles, IDs), not secrets.
4. Update nav links and page copy.
5. Deploy to GitHub Pages.
